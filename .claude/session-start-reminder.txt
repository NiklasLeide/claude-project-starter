Definition of Done — verified at sprint close:
  1. Code works (manually verify happy path)
  2. Tests written (new behavior has coverage)
  3. Existing tests pass (run the test command, zero failures)
  4. docs/CHANGELOG.md updated (one line per logical change)
  5. ./commit.sh "message" run successfully

Every sprint plan ends with a "Run DoD review" task — use the
dod-reviewer sub-agent when you reach it.

Full conventions: @.claude/shared-conventions.md
